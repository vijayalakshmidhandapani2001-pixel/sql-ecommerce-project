# E-Commerce Database Project using SQL

## 📌 Project Overview

This project demonstrates the design and implementation of an E-Commerce database using SQL. It includes multiple related tables such as customers, products, orders, and payments.

This project simulates a real-world e-commerce system and showcases my ability to design relational databases and analyze business data using SQL queries.

## 🛠️ Technologies Used

* SQL Server

## 🧱 Database Tables

* Customers
* Products
* Orders
* Payments

## 🔗 Features

* Relational database design using primary and foreign keys
* SQL queries using JOIN, GROUP BY, and aggregation
* Business insights such as total revenue, customer behavior, and product performance

## 📂 Project Structure

* `e-commerceDB.sql` → Database schema and SQL queries
* `ecommerce-portfolio.pdf` → Project documentation
* Screenshots → Query outputs and ER diagram

## 📊 Sample Queries
